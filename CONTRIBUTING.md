# Contributing to simpler-grants-gov

Thank you for your interest in contributing to the simpler-grants-gov project! We welcome contributions from the community and are excited to work with you.

## Getting Started

Before you begin, please read through this guide to understand our contribution process and requirements.

## Ways to Contribute

### Bug Reports

If you find a bug, please create an issue using our bug report template. Include as much detail as possible to help us reproduce and fix the issue.

### Feature Requests

We welcome feature requests! Please use our feature request template and provide detailed information about the proposed feature and its use case.

### Code Contributions

We accept code contributions through pull requests. Please follow these steps:

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests if applicable
5. Submit a pull request

### Bounty Program

We offer bounties for specific issues and features. Look for issues labeled with "bounty" to find opportunities to earn rewards for your contributions.

#### Getting Help with Bounties

If you have questions about our bounty program or need help with a specific bounty issue, please visit our [Bounty Program Discussion category](https://github.com/HHS/simpler-grants-gov/discussions/categories/bounty-program) where you can:

- Ask questions about bounty requirements
- Get clarification on technical specifications
- Connect with maintainers and other contributors
- Share progress updates

This is the official support channel for all bounty-related questions and discussions.
